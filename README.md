# Премиум-Лендинг Кондитерской «Nazcake» (г. Атырау)

Современный, быстрый и отзывчивый одностраничный сайт-лендинг для кондитерской/пекарни **Nazcake** (Nazcake onimderi). Создан с нуля по индивидуальным визуальным требованиям заказчика:
* Чистый белый фон и высокая контрастность (светлая тема).
* Шапка с логотипом слева и меню справа.
* Изящная пара шрифтов: геометрический `Montserrat` для заголовков и элегантный `Cormorant Garamond` для акцентных подзаголовков.
* Волнистые переходы между секциями (SVG).
* Карточки товаров без фона с удобным выбором количества (+/-) и 3D-кнопками малиново-розового градиента.
* Интерактивное окно предпросмотра товара с фото, описанием и подробным составом.
* Кастомизатор Bento-тортов в реальном времени (SVG-модель).
* Встроенный калькулятор Яндекс.Доставки по Атырау на базе Nominatim API.
* Оформление заказов и отправка заявок напрямую в WhatsApp менеджера.

## Как запустить проект локально

Проект разработан на чистом HTML, CSS и JavaScript, поэтому не требует установки сложных зависимостей:
1. Просто откройте файл `index.html` в любом браузере (двойным кликом).
2. Для корректной работы Яндекс.Калькулятора (геокодирование) и отправки запросов рекомендуется запускать через локальный сервер, например расширение **Live Server** в VS Code, или с помощью команды в терминале:
   ```bash
   npx serve .
   ```
